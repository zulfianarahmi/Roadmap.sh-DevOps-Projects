# Log Archiver Tool

## Overview
A simple CLI tool to compress and archive logs, keeping systems clean and organized while saving storage space.

## Features
- Compress logs from a specified directory into `.tar.gz`.
- Save archives in a dedicated directory.
- Log the archive operation date and time.

## Benefits
- **Save Space**: Minimize disk usage.  
- **Stay Organized**: Structured log management.  
- **Automate Cleanup**: No manual intervention needed.  

## Usage
Run the tool with the log directory as an argument:  
```bash
log-archive <log-directory>
